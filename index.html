The Monster Maker! Final Project for Iron Yard

Project Description

For my final project, I created a Monster Creator and Card Maker app in ReactJS. The idea stemmed from a "paper doll concept", where the user could assemble a creature made from different body parts. The creature could then be re-sized and animated using Fabric.js, a JavaScript HTML5 canvas library. The user is also able to search for a background image for their creation, calling out to several photo image APIs such as Pixabay, Unsplash, Pexel, as well as animated gif backgrounds through Giphy.

Project Details

  • Restful API Calls to Various Image Sites
  • ReactJS
  • Fabric.js
  • CSS Animation
